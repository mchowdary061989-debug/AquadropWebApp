# AquaDrop v2 - Vite + React + Tailwind (LocalStorage prototype)

This build includes:
- Editable Customer modal (add/edit) with service history inside modal
- Editable Inventory modal (add/edit/delete)
- Service Form modal to record service and parts used (deducts inventory)
- Sample data included for quick testing (customers, inventory)

## Quick start
1. Unzip the project folder
2. `cd` into project directory
3. Run `npm install`
4. Run `npm run dev`
5. Open the URL (usually http://localhost:5173)
